## Default Permission

Default shared buffer permissions

#### This default permission set includes the following:

- `allow-create-shared-buffer`
- `allow-read-shared-buffer`
- `allow-write-shared-buffer`
- `allow-close-shared-buffer`
- `allow-create-shared-channel`
- `allow-dispatch-shared-channel`
- `allow-close-shared-channel`
